A custom plugin for managing HOA features with a user-friendly homepage builder.

== Description ==
HOA Horizon provides a custom solution for Homeowners Associations, including a homepage builder, announcements, and front-end admin access. It allows HOA administrators to create and manage a dynamic homepage with customizable blocks, such as welcome messages, weather, announcements, and upcoming events, while offering a secure login and admin dashboard for authorized users.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/hoa-horizon` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= How do I customize the homepage? =
Use the Homepage Builder in the admin dashboard or front-end admin to drag and drop blocks, edit content, and adjust settings like theme and layout.

= Can I restrict access to the front-end admin? =
Yes, you can specify allowed roles and users in the General Settings tab of the HOA Horizon settings page.
